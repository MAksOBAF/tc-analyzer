# 📊 Telegram Chat Analyzer Pro

### 🤖 The AI Experiment: From Zero to Release in 48 Hours
This project started with a bold challenge: **Can a person with absolutely zero coding knowledge create a fully functional, cross-platform desktop application in just 2 days using only AI?** Spoiler alert: Yes, they can. 

Every line of code, every UI element, the compilation process for Windows, and even the cloud build for macOS via GitHub Actions were crafted entirely through dialogue with an AI assistant. This repository is living proof that AI is bridging the gap between an idea and a working product.

### 💡 About the Project
**Telegram Chat Analyzer Pro** is a powerful desktop tool that takes your exported Telegram chat history (JSON) and turns it into beautiful, interactive infographics. 

**Key Features:**
* 📈 **Visual Statistics:** Track message activity by days, hours, and months.
* 👥 **Top Contributors:** Find out who talks the most in your group chats.
* 🎨 **Modern UI:** Built with CustomTkinter for a sleek, dark-mode experience.
* 🌍 **Cross-Platform:** Available as a Windows Installer (.exe) and a macOS App (.app).

* ### 🇬🇧 English:
1. Open **Telegram Desktop** on your PC.
2. Go to the chat or group you want to analyze.
3. Click the three dots (menu) in the top right corner -> **Export chat history**.
4. **Crucial:** Uncheck all media (photos, videos) for a faster export.
5. **Important:** Change the format from HTML to **JSON**.
6. Click "Export". Use the resulting `result.json` file in this app.

### 💡 О проекте
**Telegram Chat Analyzer Pro** — это мощный инструмент, который превращает скучные выгрузки чатов Telegram (в формате JSON) в красивую и понятную инфографику.

**Главные фичи:**
* 📈 **Визуализация активности:** Графики сообщений по дням, часам и месяцам.
* 👥 **Топ участников:** Узнайте, кто самый активный собеседник в чате.
* 🎨 **Современный дизайн:** Стильный темный интерфейс на базе CustomTkinter.
* 🌍 **Кроссплатформенность:** Доступно в виде установщика для Windows (.exe) и приложения для macOS (.app).

* ### 🇷🇺 Инструкция:
1. Откройте **Telegram Desktop** на компьютере.
2. Выберите нужный чат или группу.
3. Нажмите на три точки (меню) в верхнем правом углу -> **Экспорт истории чата**.
4. В появившемся окне **обязательно** снимите галочки со всех медиафайлов (фото, видео), чтобы экспорт прошел быстро.
5. **Важно:** В поле "Формат" выберите **JSON** (не HTML!).
6. Нажмите "Экспортировать". Полученный файл `result.json` выберите в программе.
